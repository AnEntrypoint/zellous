// what it is

push to talk opus voice webrtc mesh, sfu hub election once you hit 3 peers, ~85ms chunks
protocol nostr nip-28 channels, nip-07 extension auth, kind-30078 voice signaling
surface browser no install, no app store, no native. pages that render and run.
backend — none — static html, vendored js, public relays. nothing to host.
