Zellous Embedding Test Harness

This harness tests Zellous embedding in 5 scenarios. All tests run locally with no external dependencies. Verify no console errors, full functionality (voice, chat, file upload), and proper state isolation.

Test 1: Same-origin (same path) pending

Embed nostr-chat/index.html in docs/test-embed-harness.html. Parent and iframe at same origin. Verify full voice + chat functionality.

Test 2: Same-origin (different path) pending

Embed via root test-embed-alt.html. Iframe loads from docs/nostr-chat/index.html with relative import paths. Verify imports resolve correctly.

Test 3: Narrow iframe (400px) pending

Constrain iframe to 400px width. Verify UI reflows: server list becomes sidebar, chat area responds, no horizontal overflow, modal max-width respects container.

Test 4: Multiple iframes pending

Two iframes on one page. Verify state is isolated per frame (each has own window.__zellous namespace). No collision or shared state.
